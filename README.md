# VIX水準に基づくS&P500底値の統計的検証 (ANOVA & Tukey HSD)

## 1. プロジェクトの背景と目的
私自身の投資経験において、VIX（恐怖指数）が急騰するパニック相場でS&P500へ資金を投下することで高い超過収益（年率20%超）を得た経験がある。
しかし、これが「単なる生存バイアス（たまたま底値だった）」なのか「統計的に有意な再現性を持つ投資行動」なのかを客観的に評価するため、過去34年分のデータを用いてバックテストおよび統計的仮説検定を行った。

## 2. 使用データと検証手法
- **データソース:** Yahoo Finance API (`yfinance`)
- **取得期間:** 1990年1月1日 〜 2024年1月1日（約8,500営業日）
- **対象指標:** S&P500 (^GSPC), VIX (^VIX)
- **分析手法:**
  - VIXの日次終値を4つの群（Normal < 20, Alert 20-30, Fear 30-40, Panic > 40）に分類。
  - 各営業日から「半年後（126営業日後）」のS&P500の収益率を算出し、群間比較を実施。
  - 第一種の過誤（多重検定問題）を防ぐため、**一元配置分散分析（ANOVA）**を実行後、有意差が確認されたため**TukeyのHSD検定**による事後検定を実施。

## 3. 分析結果

### 3.1 探索的データ分析（箱ひげ図）
![S&P500 6-Month Future Returns by VIX Phase](boxplot.png)

グラフから、VIXの水準が上がる（パニック度が増す）につれて、半年後のリターンの中央値および上振れ幅が大きくなる傾向が視覚的に確認できた。

### 3.2 統計的検定結果
- **ANOVA結果:** p-value = 9.39e-115 (< 0.05) により、フェーズ間で平均リターンに極めて強い有意差があることが確認された。

**【モデルの限界と今後の課題（高度な数理モデリングへの拡張）】**
本検証はANOVAを用いた離散的な閾値分析であり、実務的に有用なインサイトを得たが、数理モデリングの観点からは以下の拡張余地（課題）がある。

1. **連続空間への拡張と制約付き非線形回帰:**
   今回、VIXを4群に離散化（グループ分け）したことで閾値境界の情報が欠落している。これを解決するため、VIXを連続変数のまま扱う一般化加法モデル（GAM）や平滑化スプラインへの拡張を検討している。その際、過学習を防ぐ微分のペナルティ項や、金融のドメイン知識に基づく単調性制約（Monotonicity Constraint）を組み込み、より厳密で実用的な曲線を推計する。

2. **時間軸の動的評価（生存時間解析の導入）:**
   今回は「半年後」という静的なスナップショットで評価したが、実際のファンド運用では最大ドローダウンや「いつ反発するか」という不確実性のコントロールが命題となる。今後はこれを解決するため、生存時間解析（サバイバル分析）を応用し、「エントリーから+10%到達」をイベントと定義した動的な期待値推計モデルへと進化させたい。
